# MEMBERHUB (React + Vite + Bootstrap)

- React + Vite
- Bootstrap 5 (via CDN)
- Local mock data (no backend)
- Reusable components: `Button`, `Input`, `Modal`, `Navbar`, `SidebarNav`, `ReportFiltersModal`

## What is included

- /index.html — Vite entry referencing Bootstrap CDN
- /package.json — minimal metadata
- /src — application source
  - /components — reusable UI components
  - /pages — pages for each flow and screens
  - /assets — placeholder folder for images/icons
  - /layouts — administration and sidebar layouts

## How to run locally

1. Install dependencies: `npm install`
2. Run dev server: `npm run dev`
